# Pulsenow Phase 1 Static Export

This repository contains the Phase 1 Pulsenow static web app export.

## Files

- `index.html` - app entrypoint
- `assets/` - compiled CSS and JavaScript bundle

## Deploying to Vercel

Use Vercel's static site defaults:

- Framework preset: `Other`
- Build command: leave blank
- Output directory: `.`

If connecting from GitHub, create a new GitHub repository, push this folder, then import that repository into Vercel.

## Vercel Project

The GitHub repository is connected to the `pulsenow-static` Vercel project.
